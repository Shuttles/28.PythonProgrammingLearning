# 2.string

1. 单引号、双引号、三引号引起来的都是字符串。

2. 只有不同类型的引号之间可以嵌套。

3. 只有三引号支持换行！因此可以用来写注释。

4. 在一个字符串前加一个`r`或者`R`就表示后面这个字符串不转义！

   ```python
   print(r"\nHello world!")
   ```
